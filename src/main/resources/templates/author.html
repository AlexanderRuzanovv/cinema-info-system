Опыт работы с технологиями проекта

В процессе выполнения курсового проекта был получен практический опыт разработки веб-приложений на Java 17 с использованием стека Spring Boot, Spring Security, Spring Data JPA и Hibernate. Освоены принципы построения многоуровневой архитектуры приложения, реализации CRUD‑операций и работы с базой данных.

Также отработаны навыки разработки пользовательского интерфейса с применением Thymeleaf и Bootstrap 5, настройки сборки проекта с помощью Maven, а также базовые аспекты настройки безопасности и разграничения прав доступа пользователей.
