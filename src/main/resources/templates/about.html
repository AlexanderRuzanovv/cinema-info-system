Информационно-справочная система кинотеатра

Курсовая работа

О проекте

Данная система представляет собой веб-приложение для управления кинотеатром. Система позволяет вести учёт фильмов, жанров, киностудий и билетов.

Технологии
  • Java 17 - язык программирования
  • Spring Boot 3.2 - фреймворк для создания веб-приложений
  • Spring Security - аутентификация и авторизация
  • Spring Data JPA + Hibernate - работа с базой данных (ORM)
  • H2 / PostgreSQL - база данных
  • Thymeleaf - шаблонизатор для веб-страниц
  • Bootstrap 5 - CSS фреймворк
  • Maven - система сборки
Функциональность
  • Регистрация и авторизация пользователей
  • Разделение ролей (Зритель, Кассир, Менеджер, Администратор)
  • CRUD операции для фильмов, жанров, киностудий
  • Управление билетами с различными статусами
  • Поиск и фильтрация данных
  • Сортировка записей по различным критериям
  • Административная панель для управления пользователями
